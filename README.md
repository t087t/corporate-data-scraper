# corporate-data-scraper
* 本リポジトリは、企業口コミサイトから数万件規模の企業データを自動で収集し、分析可能な構造化データ（CSV形式）に変換する一連のパイプラインを提供します。

## コンセプト
* 業界ごとの年収、残業時間、満足度の傾向を定量的に比較・分析するための独自のデータセット構築を目的として開発しました。
* クローリング・スクレイピング初心者として、1,000ページを超えるランキングと、それに紐づく数万件の企業詳細ページを効率的に取得することを目指しています。

## 主な機能
#### 1. クローリング (`crawling.py`)
* ランキングページを巡回し、企業ごとのユニークIDを抽出。
* 各企業の詳細ページHTMLをローカルに一括保存、取得済みファイルのスキップ機能により重複を防止。

#### 2. スクレイピング (`scraping.py`)
* 保存済みHTML群をパースし、企業名、年収、残業時間、有休消化率などを抽出。
* レーダーチャート等の動的要素を含む数値データを正規表現を用いてクレンジング。
* 欠損値を適切に処理（`None`）し、CP932（Excel対応）形式でCSV出力。

## 技術スタック
* **言語**: Python 3.x
* **ライブラリ**: 
    * `BeautifulSoup4` (HTMLパース)
    * `requests` (HTTP通信)
    * `pathlib`, `csv`, `re` (標準ライブラリによるファイル・文字列操作)

## 使用方法
1. **環境構築**:

```Bash
pip install -r requirements.txt
```
2. **データ収集**:
`crawling.py` を実行して、企業詳細ページのHTMLをローカルに保存します。

```Bash
python crawling.py
```

3. **データ抽出**:
`scraping.py` を実行して、保存されたHTMLからデータを抽出しCSVを生成します。

```Bash
python scraping.py
```

## 動作の仕組み
1. **データ収集**: `crawling.py` を実行。指定されたページ数のランキングから企業URLを特定し、HTMLを蓄積します。
2. **データ抽出**: `scraping.py` を実行。蓄積されたHTML群を解析し、属性情報を抽出します。
3. **データ正規化**: 抽出した文字列から単位や記号を除去し、数値データとして正規化します。
4. **出力**: 分析用データセットとして `company_data.csv` を生成します。
